# CivicGuardian AI

AI-powered Government Benefits Navigator built using Multi-Agent Architecture.

## Problem

Millions of citizens struggle to discover government schemes, understand eligibility requirements, and collect the right documents.

## Solution

CivicGuardian AI acts as a personal government case worker.

Users can:

- Discover eligible schemes
- Upload supporting documents
- Detect missing paperwork
- Receive personalized action plans
- Generate downloadable reports

## Architecture

Multi-Agent System:

- Coordinator Agent
- Eligibility Agent
- Scheme Agent
- Explainer Agent
- Document Agent
- Planner Agent
- Report Agent

## Features

✅ Scheme Recommendation

✅ Eligibility Analysis

✅ Document Intelligence

✅ Missing Document Detection

✅ Personalized Action Plans

✅ PDF Report Generation

- Python
